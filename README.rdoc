
= AngularJS with RoR test app
== Test app by David Bryant Coperland for the "AngularJS with Ruby on Rails" tutorial.

Deployed to https://receta-testapp.herokuapp.com

To run locally:

* clone it
* bundle install
* rake db:setup
* rails s

To run tests:

JS tests:
* rake teaspoon
Specs and browser tests:
* rspec
